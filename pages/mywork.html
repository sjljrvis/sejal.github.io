mobyStore



mobyStore is simple a simple RESTful service to manage online mobile store.

S.P.A.C.E



S.P.A.C.E is a simple android app with 3 functionalities

  • Lets user translate spoken text from English to German
  • User can control laptop/Desktop mouse cursor from mobile( Destination IP address is needed and receiver machine must run space.jar ) ,here app gets access to mouse cursor by socket binding (app is binded to space.jar via IP address:port for eg 192.168.1.101:5555)
  • Converstion mode lets user to talk with S.P.A.C.E ,where app responds to user queries.

</>Technology stacks used :- JAVA, Android ,google Speech API , Microsoft translator API


Future upgrades
  • Improvements in UI
  • Integration with wit.AI bot kit to make conversation mode realistic
